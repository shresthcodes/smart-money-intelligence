## ❌ Common Problems

### Problem 1: "python is not recognized"

**Solution:** Python install nahi hai
- Python 3.8+ download karo from python.org
- Install karte waqt "Add Python to PATH" tick karo

### Problem 2: "No module named 'streamlit'"

**Solution:** Streamlit install nahi hua
```bash
python -m pip install streamlit
```

### Problem 3: Terminal band ho gaya

**Solution:** Phir se kholo aur command run karo!
