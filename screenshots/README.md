# Dashboard Screenshots

This directory contains screenshots of the Smart Money Intelligence Platform dashboard.

## How to Add Screenshots

1. Run the dashboard: `streamlit run dashboard/app.py`
2. Navigate to each page
3. Take screenshots of:
   - Market Overview page
   - Institutional Activity page
   - Sector Analysis page
   - Predictions page
4. Save screenshots with these names:
   - `market_overview.png`
   - `institutional_activity.png`
   - `sector_analysis.png`
   - `predictions.png`

## Screenshot Guidelines

- Use full browser window
- Ensure charts are fully loaded
- Include the sidebar navigation
- Use consistent browser zoom level (100%)
- Recommended resolution: 1920x1080 or higher
